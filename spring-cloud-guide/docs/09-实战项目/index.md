- 设计微服务架构

## 🏗️ 项目架构

```
电商微服务系统
├── eureka-server      # 服务注册中心
├── config-server      # 配置中心
├── api-gateway        # API 网关
├── user-service       # 用户服务
├── product-service    # 商品服务
├── order-service      # 订单服务
└── payment-service    # 支付服务
```
